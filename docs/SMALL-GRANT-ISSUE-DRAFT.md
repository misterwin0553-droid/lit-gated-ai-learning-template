# Small Grants: Lit-Gated Interactive Learning Template

### 1. What is your project? (max 100 words)

I’m building an open-source Lit-gated interactive learning template. The reference demo uses NASA Mars materials to show progressive access across public learner content, student clues, teacher guides, source annotations, and AI assistant context gating. The goal is to teach developers how to apply Lit access control to layered educational content, not to build a commercial education product.

Repo: TBD
Demo: local MVP currently includes executable mock mode, interactive role/task demo page, NASA source note, Lit SDK dependencies, and a browser wallet Lit unlock route for the protected student layer.

### 2. How is Lit used for this project? (max 100 words)

Lit will be used to encrypt non-public learning layers and enforce access conditions for student, teacher, and source/creator materials. The demo will show how public content remains open while clues, answer keys, source notes, and AI task contexts are unlocked only for authorized roles. Lit is the core mechanism for progressive disclosure and role-aware AI context access.

### 3. How will you improve your project with this grant? What steps will you take to meet this objective? (max 200 words)

Grant support will fund the transition from a working mock template to a complete Lit developer tutorial. Planned work:

1. Extend the current browser wallet Lit unlock route from the student layer to teacher/source layers and polish the authorization flow.
2. Harden the NASA Mars reference demo with final citations and source notes.
3. Polish the interactive role/task demo and access report.
4. Expand the step-by-step tutorial and architecture guide into a complete developer walkthrough.
5. Record a short walkthrough showing how to adapt the template to another learning module.
6. Publish weekly updates and a final grant report.
