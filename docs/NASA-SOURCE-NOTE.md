## Why this image fits the reference demo

The demo asks learners to reason from visible geological evidence rather than jump to a conclusion. A valley-network image is suitable because it supports layered educational prompts:

- public layer: observe surface shapes;
- student layer: identify branching and flow-like structures;
- teacher layer: distinguish observation from proof;
- source layer: cite image source and explain content-design decisions.
